Page d'installation
Cette page indique que les fichiers sont en place. Suivez les étapes ci-dessous pour finaliser l'installation.
1

Créer la base de données

Ouvrez phpMyAdmin et créez une base nommée reception_salfa, puis importez le fichier import_full.sql depuis le dossier wamp_deploy/database/.

2

Configurer Apache

Copiez reception-salfa.conf vers C:\wamp64\conf\extra\ et ajoutez Include conf/extra/reception-salfa.conf dans httpd.conf. Redémarrez WAMP.

3

Lancer l'application

Exécutez deploy.bat ou accédez directement à l'application via le bouton ci-dessous.
